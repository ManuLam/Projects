Steps for using the application

Note: Must have npm / nodemon installed 

1. open the command prompt
2. cd into "project" directory
3. have npm installed as well as nodemon
4. use code "nodemon app.js"

Once "app.js" is running

1. type in "http://localhost:4000/index.html"

Folder Information
models		- mongoDB database models
node_modules	- npm modules
public		- MAIN FOLDER
Removed		- removed files / code
TestJunk	- used to test mongoDB / website
